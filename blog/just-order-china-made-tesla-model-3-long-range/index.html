刚刚下订了国产 Model 3 长续航版, 我的心理历程

我的心理历程-1

其实这不是我第一次下订 Model 3 了.

第一次下定是 2017 年 6 月, 那时候 Model 3 全球都还没上市. 首批 30 辆车的交付是 2017 年 7 月, 同年 12 月才开始向普通消费交付.

当时国内的订金是 8000 元人民币, 我毫不犹豫地下定了. 特斯拉对热爱数码科技的我有着极大的吸引力, 3 之前的 S 和 X 都是上百万起步的车型, 简直遥不可及, 每次在网上看到 S 和 X 的文章/视频/网友的点评都只有羡慕的份, 现在终于有一款我可能买得起的特斯拉了, 别提当时的我有多兴奋和期待. 我相信很多人都有过类似的心理体验, 不一定是车, 也可以是手机电脑相机等等.

Model 3 刚开始交付后的一段时间, 我每天上网冲浪, 都会找找今天有没有新的 Model 3 视频可以看. 我的云体验告诉我, Model 3 的设计真的太酷了, 嘴巴没有进气口, 中控台只有一块显示屏 (当然还有方向盘), 长这么大没见过这样设计的车, 真是活久见了.

但持续关注了两三个月之后, 我对 Model 3 的新鲜感开始下降, 逐渐对出现在我信息流中的 Model 3 的视频和文章感到索然无味, 唯一还能刺激我疲软的神经的是 Model 3 何时才能在国内交付, 这类小道消息断断续续刺激了我一年多.

终于, 2018 年 11 月 16 日, 有网友表示他们突然收到了特斯拉发来的 Model 3 邮件, 通知他们选配交钱, 准备提车!

我一下子就兴奋起来, 等了一年半, 终于来了! 但是一看价格, 心凉了一半. 我期望购买的长续航版售价高达 58.8 万, 比我预期的 35-40 万之间超了大概 20 万, 顿时感到心灰意冷, 开心得太早了, 以为自己真的买得起特斯拉…

感到心灰意冷的肯定不止我一个, 才不到一周, Model 3 降价了, 长续航降了 4.8 万至 54 万.

降价还没有结束, 2018 年 12 月长续航又降了 4.1 万, 至 49.9 万, 还是太贵了! 这时 Model 3 的国内交付甚至还未开始, 价格就已连降两次.

49.9 万这个长续航版是”四驱"长续航版, “后驱"长续航国内价格当时还没公布, 于是心灰意冷的我还没有绝望, 我等! 我等后驱长续航, 哼!

2019 年 2 月 1 日, 后驱长续航在国内上市, 售价 43.3 万. 这个价格和我 40 万以内的预期差不太远, 但当时 Model 3 需要购置税, 选配我喜欢的白色车身貌似还要加两万, 加上保险等费用, 落地也将近 50 万了… 真的好鬼贵…天啊...

不瞒大家, 这时候我有点绝望了, 所谓的长续航也只不过能开 500 公里, 选择长续航是我的底线, 买短续航? 不可能的.

2019 年 2 月 22 日, 首批 Model 3 在国内交付. Model 3 第一批后驱长续航车主以 43.3 万的裸车价格, 将近 50 万的落地价格, 成为国内第一批开上 Model 3 的幸运儿.

我以为后驱长续航短期内不会再降价, 但我错了, 2019 年 3 月 1 日, 后驱长续航降价 2.6 万至 40.7 万. 第一批幸运儿自嘲成了 Model 3 首批韭菜. 但这时候我还是买不起, 连做韭菜的资格都没有.

令人意外的是, 大概三个星期后, 后驱长续航竟然涨价了, 更令人意外的是, 三个星期后又涨了一次! 后驱长续航的最终售价定格在 42.1 万. 不过我依然买不起.

这时我已经对进口版不抱希望了, 由于特斯拉 2018 年 7 月宣布在上海建厂, 我把最后的希望放在了国产版.

在我彻底绝望的 2019 年 4 月 (因为这时候后驱长续航价格不降反升, "还会降价"的美好愿望破灭), 有媒体报道特斯拉上海工厂正在正搭建钢结构屋顶, 可见此时离投产还远.

于是我迫不及待, 但不得不耐住性子地继续等. 在等待的过程中, 我不断看到车处们对 Model 3 各种小问题的吐槽, 当然也有很多真香警告, 接着是提车 3 个月的体验, 提车 6 个月的体验, 真实车主们对 Model 3 的新鲜感也开始下降, 评价在我看来慢慢变得更理性了一些.

我理解的 Model 3

关注了这么久, 现在 Model 3 给我的印象大概是这样的…...

科技感依然很强, 像给手机升级系统那样给 Model 3 升级系统很值得期待, 因为每次升级可能都会带来一些好玩好用的东西.

对于超级简洁的中控台, 从超级喜欢变成现在的一般喜欢. 关注了一些豪华内饰的车型之后, 我开始理解为什么有人把这种简洁称为光秃, Model 3 至少在视觉上无法带给用户一种豪华舒适的体验. 当然一般喜欢依然是喜欢, 简洁和豪华是通往可用性的两种不同的途径, 没有优劣之分, 我可以同时喜欢简洁和豪华.

Model 3 驾驶座不少车主表示舒服, 但很少车主愿意肯定后排座椅的舒适性. 年纪越大, 越是觉得 "花三四十万买一辆所谓的豪华车 (听说 30 万以上被定义为豪华车), 却无法让家人舒适地坐在后排” 是一种自私的行为. Model 3 后排座椅低, 坐垫小, 远途出行应该会觉得累.

Model 3 的玻璃车顶, 我从超级喜欢转变为有点担心. 广东的夏天经常十分闷热, "坐着都出汗" 并不是夸张的说法.

Model 3 作为电车维护保养成本很低, 2 万公里可能只要换一个空调滤芯. 至于电费, 虽然便宜, 但除非自己有桩, 电费不一定低于燃油车的油费. 有一次搭滴滴, 司机说他的双擎雷凌百公里 3.2 升的油耗, 即不到 20 元的油费. 作为对比, Model 3 百公里 12 度电算是比较好的成绩, 有些车主可能去到 15 甚至更高, 12 度电在外面充电要 15-20 元.

Model 3 作为电车, 其加速性能很强, 但我开车比较佛系, 我买 Model 3 的原因从来不是因为它加速快.

Model 3 图片真的很好看, 但偶尔看到的在路上跑着的实车似乎颜值一般, 而且偏小, 尤其是这个小, 验证了车主们的 "Model 3 只适合一两个人用, 家庭用车并不适合” 的说法.

Model 3 的自动驾驶可能并没有宣传的那么厉害. 特斯拉的全自动驾驶需要单独付费购买, 国内上线遥遥无期, 至于国产 Model 3 自带的基础辅助驾驶, 只能 "根据其他车辆与行人在行驶车道内自动辅助实施转向、加速和制动”. 关键词: 在行驶的车道内. 所以基础自动驾驶不能自动变道, 不能自动超车, 更不能识别并根据红绿灯停止或启动. 所谓的 “自动转向” 只是当车道是弯道时, 自动沿着车道转动方向盘.

特斯拉宣传的全自动驾驶是什么意思呢? 就是告诉它目的地之后, 你就不需要任何操作, Model 3 会自动规划最优的路线, 然后自动执行每一个超车变道, 上高速变道, 下高速变道, 自动转向, 自动掉头, 自动识别信号灯并自动作出反应等等. 这些功能遥遥无期, 不要相信马斯克关于全自动驾驶功能上线时间的预估.

不过呢, 很多车主表示 Model 3 的基础辅助驾驶体验就够用了, 尤其是高速路段. 虽然变道时, 上高速时, 下高速时, 都需要司机提前观察路况, 提前作出判断, 然后手动执行决策, 但如果只是单调地沿着本车道行驶, 交给自动驾驶会轻松很多.

国外评测机构显示, Model 3 的安全评级很高, 是最安全的车型之一.

以上是我下订 Model 3 之前, 对 Model 3 的大概印象, 只是云印象.

我的心理历程-2

重新回到 Model 3 的订购上吧…

前文讲到从 2019 年 4 月开始, 我已经不考虑进口版 Model 3 了, 而当时上海工厂才开建四个月, 对于还要等多久, 真的毫无头绪.

但建设速度出现意料的快, 2019 年 5 月 31 日, 国产 Model 3 开放预订. 虽然交付时间为 6-10 个月, 但至少让人看到了一丝希望. 可惜这次开订的只有短续航版本, 没有我期望购买的长续航.

随后的半年多时间里, 国产 Model 3 获得了国家新能源汽车补贴, 免了购置税,车漆也降价, 并标配了基础版自动驾驶, 在这之前, 即使基础自动驾驶也要单独付费购买. 一系列的政策显著降低了 Model 3 的售价.

2020 年 1 月 3 日, 国产 Model 3 短续航版售价最终确定为 29.9 万, 落地价只要再加 8000 车漆 (如选配) 和几千保险, 即 30 万出头. 1 月 7 日, 首批国产 Model 3 向普通消费者交付.

这时候其实我有点动心了, 但真的很介意短续航版满电只能开 380 公里的事实, 于是我选择继续等. 上海工厂肯定会生产长续航版本, 而且不会等太久, 这是基本的商业常识.

2 月 12 日, 有网友发现工信部公示了最新的一批申报车型, 其中就有国产长续航 Model 3. 很多人根据进口短续航和进口长续航的价差, 推断国产长续航的价格将在 35 万以内, 这也符合我 2017 年 6 月的价格预期.

4 月 7 日, 有消息称长续航将于本周开放预订.

4 月 9 日, 有消息称国产长续航售价 33.9 万.

4 月 10 日, 即今天, 特斯拉官网公布国产长续航价格, 并开放预订, 售价比传言的 33.9 多了 50 元.

要订购吗? 当这一天真的到来, 我反而有点犹豫.

从 2017 年 6 月到今天, 等了将近三年, 我对 Model 3 的新鲜感早已消失, 即使提车了恐怕也不会带给我多少兴奋. 三年前, 我完全可以接受 Model 3 是一款大号电子产品或者电子玩具, 而现在, 我开始考虑花三十多万重金买的车却不能让家人坐得舒适, 出行时不能多带点行李, 只是为了特斯拉的科技感, 为了满足我个人的喜好, 这会不会有点自私, 会不会不值.

Model Y 已经在美国上市, 上海工厂已经在建设 Y 的生产线, 有消息称国产 Y 第三季度就可以下线. 国产长续航 Model Y 肯定不会比国产长续航的 Model 3 贵多少, 但作为 SUV, 车内空间完胜, 是一款实用的家庭用车. 但如果继续等 Y, 还要至少等半年.

可是我已经等了三年, 等不动了, 于是我一边犹豫, 一边再次下定 Model 3, 但这一次是国产 Model 3… 希望提车后它能带给我一些惊喜…
