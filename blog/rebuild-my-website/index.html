网站重构

作者: 科技微讯

日期: 2019/09/01

网站正处于施工中, 不知道什么时候修好, 受够了 Wordpress 啦, 也受够了阿里云... WP 是 PHP 写的, 我不想学 PHP 了, 会一点 js, 打算自己用原生 html css 写一些简单的静态网页, 可能再加一点点 js.

科技微讯这个网站差不多 5 岁了, 大概发布了 100 篇文章, 有数百条评论吧, 其实后台有评论的统计数据, 不过现在我无法登录后台, 对这种情况习惯了, 之前还试过大半年无法登录后台, 一直不知道怎么解决.

更早之前, 大概是 17 年, 有很长一段时间这个小站点的服务器频繁死机, 始终无法定位原因, 向阿里云售后提了好几次工单, 对方的回复就是让我升级服务器, 升了一次还是死机, 没有任何改善, 对方机械性地建议我再升级, 于是我又升了, 还是死机, 好像升了三次? 不敢再升, 因为服务器费用对我这个小站点而言太高了. 每次死机我就重启服务器, 重启一次大概能坚持几个钟头, 然后再死机, 再重启. 不记得持续了多久, 后来终于很少死机了, 但这段经历让我对阿里云失去了所有好感.

当时我已经在用 dnspod 的域名解析, 后来 dnspod 被腾讯收购, 于是我被动地开始使用腾讯云的服务, 期间向腾讯云提过几次工单, 对比过才知道腾讯云的客服比阿里云好得多, 更耐心, 更专业, 而且至今没有收到腾讯云的销售电话, 阿里云我已经接到过不下 5 次, 售前售后都在推销产品, 让人怀疑当年无缘无故死机是不是和阿里云有关系...

我在阿里云的租期结束啦, 受够了 Wordpress, 也受够了阿里云, 所以决定扔掉这两个包袱, 重新开始~
